Corner Detection

  1. Method Description.

    For this problem, I first convert the image to grayscale image, then I compute the gradient of the image. From the gradient, I can calculate different term in MMM matrix. For example, for ∑w(x,y)Ix2\sum w(x,y)I_{x}^2∑w(x,y)Ix2​, I first compute the Ix2=dx2I_x^2 = d_x^2Ix2​=dx2​, where dxd_xdx​ is the gradient along xxx axis of the image. Then I convolve this image with a gaussian filter, so the result would be the desired ∑w(x,y)Ix2\sum w(x,y)I_{x}^2∑w(x,y)Ix2​.

    After getting all three terms, I can then calculate the determinent and trance of the matrix MMM. Then I can compute the response with r=det(M)−k∗(trace2)r=det(M)-k*(trace^2)r=det(M)−k∗(trace2), where kkk is a constant.

    We know from the equation derivation that any R>0R > 0R>0 could represent a corner in the image. So I threshold the response image, then use a non-maximum suppression with a certain window size to determine the corner.
