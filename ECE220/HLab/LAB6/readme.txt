This program is a implementation of Smith-Waterman Algorithm.

ECE 220 HLab 6, 10/28/17
Nickel Liang, zuodong2@illinois.edu

=========================
To compile the program:
    $ make error

To run the program with default input sequence file:
    $ make run

To run the program with other input sequence file:
    $ ./hlab6 <filename>
=========================
The input sequence file has to be continuous string of gene code

The input sequence cannot be comma or space separated.
